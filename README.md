# Scam Detector

## Overview
This project is designed to help users detect potential scams through analyzing user input.

## Features
- Improved HTML structure for better user experience.
- Advanced detection logic to identify scam-related keywords.

## Getting Started
### Prerequisites
- Basic understanding of HTML and JavaScript.

### Installation
1. Clone the repository.
2. Open `index.html` in a web browser.

### Usage
- Enter text in the input field and click 'Detect' to check for scams.
